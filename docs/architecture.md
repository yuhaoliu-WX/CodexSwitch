## 十、常见问题

### 10.1 使用 DeepSeek 时需要关闭 VPN

**现象**：切换到 DeepSeek 后，Codex 频繁显示 "Reconnect"，只有关闭 VPN 后才恢复。

**原因**：Moon Bridge 是一个本地代理（127.0.0.1:38440），它将 Codex 的请求转发到 DeepSeek 的 API 服务器（国内）。
VPN 开启时，所有流量被路由到海外出口再绕回国内，导致 DeepSeek 连接延迟高、不稳定，触发 Moon Bridge 的重连机制。

**推荐工作流**：

```
1. 开 VPN     → 启动 Codex（OpenAI 被墙，需要 VPN）
2a. 切 DeepSeek → 关 VPN（DeepSeek 直连国内，更稳定）
2b. 保持 OpenAI  → 保持 VPN
```

**理想方案（VPN 分流 / Split Tunneling）**：
在 VPN 客户端中配置 split tunneling，只让以下域名走 VPN 隧道：
- `api.openai.com`
- `oaiusercontent.com`

其他流量（包括 DeepSeek `api.deepseek.com` 和 Moon Bridge）直连本地网络。
这样全程不用手动开关 VPN。

### 10.2 托盘图标在哪里？

启动 `python main.py` 后，系统托盘区域（任务栏右下角）会出现 Codex Config Switcher 的图标。
左键点击显示窗口，右键点击弹出操作菜单。

### 10.3 如何完全退出程序？

点击主窗口的 **Quit** 按钮 → 选择 "Exit"（会自动停止 Moon Bridge 进程），
或右键托盘图标 → **Exit**。

注意：点击窗口右上角 × 只是隐藏到托盘，程序仍在后台运行。
